fonction de test et de génération d'erreur

Paramètres
msg: message à afficher
cond: condition d'erreur
errnum: numéro de l'erreur système ou 0 s'il n'y en a pas.

Cette fonction affiche le texte d'erreur msg sur la sortie d'erreur et lance une exception. Si un code d'erreur errnum différent de 0 est fourni, l'exception générée est une erreur système (std::system_error) dont errnum est le code d'erreur. Sinon, l'exception est une std::runtime_error dont msg est le message.

Par exemple le code suivant ouvre un FD en écriture en créant le fichier toto.txt et teste sa création et génère une exception système en cas d'erreur

fdout = open("toto.txt", O_CREAT|O_WRONLY, S_IRUSR|S_IWUSR);
exit_error("Création du fichier", fdout==-1, errno);
